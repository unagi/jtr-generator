# jtr-generator - ユーザー向けガイド

## できること

- 履歴書（JIS様式）と職務経歴書をPDFで作成できます
- 会話でヒアリングされるので、答えるだけで進みます
- 既存の履歴書・職務経歴書を渡して、整理し直すこともできます

## 使い方（会話の流れ）

1. 作りたいものを伝える（履歴書/職務経歴書/両方）
2. 必要な情報を順番に聞かれる
3. 内容を確認しながら整える
4. PDFとして受け取る

## 事前に用意しておくと良い情報

- 氏名、連絡先、生年月日、住所
- 学歴（入学・卒業の年月）
- 職歴（入社・退職の年月、現職ならその旨）
- 免許・資格
- 職務経歴書用の情報（役割、成果、規模、期間、使用技術など）

## 職種別の深掘りガイド

職種ごとの書き分けやキーワードは、`references/job_type/README.md` を参照してください。

## 既存書類の持ち込み（再開・更新）

- 既存の履歴書・職務経歴書を渡すだけで整理できます
- 前回の続きや更新にも使えます

## 生成結果（PDFの受け取り）

- 履歴書のみ、職務経歴書のみ、両方のいずれでも生成できます
- 出力されたPDFは会話中に案内されます

## 相談・やり直し

- 不足がある場合は追加で質問されます
- 途中の修正ややり直しもできます
