# 職種別ガイド（職務経歴書）

このディレクトリは、職種別の「深掘りポイント」と「キーワード」をまとめた詳細ガイドです。
職務経歴書の作成時は、まず全体方針を `references/career_sheet_best_practices.md` で確認し、
その後に該当する職種ガイドを参照して内容を具体化してください。

※ Agent Skills環境ではオフラインのため、外部リンク参照不可の場合があります。要点は本ガイドに要約済みです。

## 使い方

1. 職種に最も近い分類を選ぶ
2. 職務要約・職務経歴・成果指標・キーワードの観点を反映
3. 応募先のJDに合わせて強調点を調整
